Теги элементов фромы

Тег FORM <form action="обработчик" method="метод_отправки_данных" enctype="multipart/form-data"></form> - контейнер для элементов формы

атрибуты тега form:

  • action - указывает обработчик данных формы на сервере
  • method - метод, которым будут отправлены данные формы (post / get)
  • enctype="multipart/form-data" - указывается, ели в форме есть поле для загрузки файлов
  • Доступны дополнительные атрибуты: autocomplete / disabled / name
Тег FIELDSET и LEGEND <fieldset></fieldset> - группирует связанные элементы в форме, отображается в виде рамки вокруг них

Для fieldset доступны дополнительные атрибуты: name / disabled

<legend></legend> - заголовок связанных элементов формы (вкладывается в fieldset)
Тег LABEL <label for="select"></label> - текстовая метка для элемента

атрибут for - привязывает label к элементу списка по id

Теги SELECT, OPTGROUP, OPTION
<select multiple size="2"></label> - выпадающий список
<optgroup label="название группы"></optgroup> - группировка элементов списка (дополнительный элемент, вкладывается в select)
<option value="значение1" selected></option> - элемент списка (вкладывается в select или optgroup)

атрибуты выпадающих списков:

  • multiple - доступен множественный выбор
  • size - количество отображаемых элементов
  • selected - элемент отображается, как выбранный по-умолчанию
  • Для списков доступны дополнительные атрибуты: autofocus / disabled / required / name / value
Тег TEXTAREA <textarea cols="10" rows="5" wrap="soft"></textarea> - поле для ввода многострочного ввода текста

атрибуты textarea:

  • cols - чило колонок
  • rows - чило строк
  • wrap="hard|soft|off" - перенос длинного текста - перенос может сохранятся при отправке на сервер (hard), перенесенный автоматически текст будет передаваться одной строкой (soft), вместо переноса будет появляться полоса прокрутки (off)
  • дополнительные атрибуты: autofocus / readonly / disabled / required maxlength / minlength / placeholder / name / value
Тег INPUT
<input type=""> - элемент формы. Отображение зависит от значения атрибута type
* label можно привязывать к элементам на только по id, но и вклыдываю input в label, например
<label>Текстовая метка<input type=""></label>
Текстовые поля
<input type="text"> - для ввода текта
<input type="password"> - для ввода пароля (скрывает вводимый текст)
<input type="email"> - для ввода email (валидация входных данных)
<input type="search"> - для поиска
<input type="url"> - для ввода url (валидация входных данных)

Для текстовых input доступны дополнительные атрибуты: autocomplete / autofocus / placeholder / readonly / disabled / required / pattern / maxlength / minlength / name / value

Числовые поля
<input type="number"> - для ввода чилел
<input type="range"> - ползунок

Для числовых input доступны дополнительные атрибуты: autocomplete / autofocus / placeholder (только type="number") / readonly / disabled / required / pattern / max / min / step / name / value

Загрузка файлов <input type="file" multiple accept="image/*"> - для загрузки файлов

атрибуты input type="file":

  • multiple - загрузка нескольких файлов
  • accept - ограничение по типу файлов
  • дополнительные атрибуты: name / disabled / required / autofocus
Взятие даты и времени

С полями для получения даты и времени следут быть осторожнее. Они по-разному отображаются в различных браузерах и не отображается в старых браузерах. Лучше использовать js библиотеки для формирования подобных полей

<input type="time"> - получение времени
<input type="date"> - получение даты
<input type="week"> - получение недели
<input type="month"> - получение месяца

дополнительные атрибуты полей ввода даты и времени: autofocus / readonly / disabled / required max / min / name / value

Чекбоксы и радиокнопки
<input type="checkbox" checked> - одиночный флажок
<input type="radio" checked> - одиночная радиокнопка

checked - флажок или радиокнопка отображаются, как выбранные по-умолчанию

Группа флажков (данные будут отправлены на сервер массивом):
<input type="checkbox" name="checkbox_name[]">
<input type="checkbox" name="checkbox_name[]">
<input type="checkbox" name="checkbox_name[]">
Группа радиокнопок (для выбора одного из нескольких значений):
<input type="radio" name="radio_name">
<input type="radio" name="radio_name">
<input type="radio" name="radio_name">

дополнительные атрибуты чекбоксов и радоикнопок: value / name / disabled / required / autofocus

Кнопки: теги INPUT и BUTTON
<input type="button" value="Просто кнопка"> - кнопка
<input type="reset" value="Очистить"> - кнопка очистки полей формы
<input type="submit" value="Отправить"> - кнопка отправки данных формы на обработчик, указанный в action
<button type="button">Просто кнопка</button> - кнопка
<button type="reset">Очистить</button> - кнопка очистки полей формы
<button type="submit">Отправить</button> - кнопка отправки данных формы на обработчик, указанный в action
Взятие цвета <input type="color">
Скрытое поле <input type="hidden">
Описание дополнительных атрибутов:
  • autocomplete="on|off" - запоминание данных, введенных в текстовое поле
  • autofocus - фокусировка на поле при загрузке страницы
  • placeholder - подсказка
  • readonly - не позволяет изменять значение поля (выделение и копирование текста доступны)
  • disabled - отключает возможость редактирования и копирования содержимого поля
  • required - поле обязательно для заполнения
  • pattern - проверяет поле на сответствие регулярному выражению
  • maxlength - ограничивает ввод текстовых данных
  • minlength - ограничивает ввод текстовых данных
  • max - ограничивает ввод числовых данных
  • min - ограничивает ввод числовых данных
  • step - шаг при вводе числовых данных
  • name - имя, которое будет использовано для доступа к элементу (для доступа к элементу из javascript, для доступа к пользовательскому вводу или выбору на сервере)
  • value - текст, отображаемый на кнопке, в поле или связанный с полем текст (обычно, пользовательский ввод или выбор)
